Frequently Asked Questions

Common questions

Capultura is a Chrome extension that records real browser HTTP traffic and DOM actions, then exports them into JMX (for Apache JMeter) or Playwright test scripts. The BETA is for early adopters and testers — expect rough edges and occasional regressions.

Visit the GitHub repository and download the latest BETA release. Load the unpacked extension in Chrome via chrome://extensions. Check the README for detailed installation instructions.

Currently Capultura targets Chrome/Chromium (version 88+) with Manifest V3. Firefox and Edge support is on the roadmap but not yet available.
